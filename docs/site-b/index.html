site-b の
別サンプル。

こちらは 2 つ目の個別サイトです。テーマや文言を変えても、同じ共有 CSS を使って素早く増やせます。

カードを見る トップへ戻る

共通の見た目を使いながら、中身だけ別サイト用に差し替えるイメージです。

Fast 既存サイトをコピーして見出しだけ変えれば増やせます。
Safe 別サイトの HTML を触っても他のフォルダへ影響しにくい構成です。
Shared 共通の画像や CSS は `assets` に寄せられます。
Clear URL とフォルダ名が一致するので把握しやすいです。

増やす手順

一番手早いのは `site-a` か `site-b` のフォルダを複製して、名前とリンクだけ調整する方法です。

1. 複製

docs/site-b/ をコピーして docs/site-c/ を作ります。

2. 追記

トップの docs/index.html に新しいサイトへのリンクを足します。
