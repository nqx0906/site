site-a の
ダミーページ。

これは `docs/site-a/index.html` に置いたサンプルです。 1 repo の中で個別サイトを増やすときは、この単位でフォルダを追加していきます。

コンテンツを見る トップへ戻る

site-a から共通画像を読むときは ../assets/... を使います。

URL /site/site-a/ として公開されます。
Shared CSS 見た目は共通の docs/assets/styles.css を使っています。
Folder 個別サイトのファイルはこのフォルダに閉じ込められます。
Reusable 別サイトを作るときはこのフォルダをコピーすれば始められます。
