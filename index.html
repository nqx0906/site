GitHub Pages 公開方法

中身がダミーでも、公開手順だけ先に通しておけるようにしています。必要になったら HTML を差し替えるだけです。

1. GitHub 側で Pages を有効化

Repository Settings の Pages で Source を GitHub Actions に設定します。

2. main に push すると自動公開

.github/workflows/deploy-pages.yml がルートの index.html を Pages へデプロイします。

このページの本文はすべてプレースホルダーです。必要になったら自由に差し替えて使えます。
